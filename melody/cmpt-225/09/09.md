# Recursion on Trees

CMPT 225

## Recursion

Recursion: A definition of a function is recursive if the body cointains an application of itself.

Consider: {% katex display %}
S(n)=\sum_{i=0}^{n} i
{% endkatex %}

or 

{% katex display %}
S(n) = \begin{cases}
0 & \text{if } n=0\\
n+S(n-1) & \text{if } n>0
\end{cases}
{% endkatex %}
